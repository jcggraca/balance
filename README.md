## Getting Started
To start using Balance, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/jcggraca/balance.git
   ```
2. Navigate to the project directory:
   ```bash
   cd balance
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
